Licznik Punktów – Koszykówka
Prosta aplikacja do zliczania punktów w meczu koszykówki. Użytkownik może dodawać 1, 2 lub 3 punkty za pomocą trzech przycisków.

Nauczasz się:
Button – obsługa kliknięć i wykonywanie akcji po naciśnięciu

OnClickListener – przypisywanie logiki do przycisków

TextView – wyświetlanie aktualnego wyniku

Zmienne i inkrementacja – dodawanie punktów do wyniku

Organizacja logiki – oddzielenie UI od działania aplikacji

Edge-to-Edge UI – dostosowanie interfejsu do nowoczesnych ekranów

Prosty projekt interfejsu – rozmieszczenie przycisków i wyniku
